三种业态对应三套分账比例——平台商家商家得75.4%，联盟商家商家得71.4%，综合商城商家得77.9%——商家的每一笔交易拿回绝大多数，平台的每一分收入都来自实际发生的交易服务费。

- 平台商家分账（汇付收单）：渠道0.6% + 商家75.4% + 平台5% + 服务站+推广者9% + 城市服务商1% + 消费金3% + 营销池4% + 风控备用金2% = 100%。
- 联盟商家分账：渠道0.6% + 商家71.4% + 平台渠道费4.5% + 平台5% + 服务站+推广者10% + 城市服务商1% + 消费金2% + 营销池3.5% + 风控备用金2% = 100%。
- 综合商城分账：渠道0.6% + 商家77.9% + 平台6% + 服务站+推广者6% + 城市服务商1% + 消费金3% + 营销池3.5% + 风控备用金2% = 100%。
- 三套比例的差异有精确的商业逻辑：联盟商家平台让利4.5%（渠道费），换取流量导入；综合商城商家拿得最高（77.9%），因为平台自营无外部推广分成。
