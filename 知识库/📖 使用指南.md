# 📖 Obsidian × Claude Code · 协同使用指南

## 快速上手

### 第一次打开 Obsidian

1. 打开 Obsidian → 选择 "打开文件夹作为 Vault"
2. 选择 `/Users/mac/Downloads/君衡 项目文件夹`
3. 在 "设置 → 第三方插件" 中关闭安全模式
4. 安装推荐插件：Dataview、Templater、Calendar、Kanban

### 每日工作流

```
早上
  ├── Obsidian: 打开 🏠 首页，查看待办和最近活跃
  ├── Obsidian: Cmd+D 新建日报，填写今日计划
  │
工作中
  ├── Claude Code: 运行生成脚本、修改文档
  ├── Obsidian: 随时 Cmd+N 新建笔记记录想法 → 存入收件箱
  │
下班前
  ├── Obsidian: 完成日报，更新进度
  ├── Claude Code: ./regenerate_all.sh 批量生成文档
  └── 自动: 变更记录写入 Obsidian 变更日志
```

### 周循环

```
周一: 查看 🏠 首页 Dataview → 了解本周焦点
周三: 整理收件箱（15分钟）
周五: 写周报（用模板），同步 Memory
```

## 核心快捷操作

### Obsidian 快捷键

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 快速打开 | `Cmd + O` | 打开/搜索任何文件 |
| 命令面板 | `Cmd + P` | 所有命令 |
| 新建笔记 | `Cmd + N` | 自动应用模板 |
| 新建日报 | `Cmd + D` | 用日报模板 |
| 插入模板 | `Cmd + T` | 选择模板 |
| 切换预览 | `Cmd + E` | 编辑/阅读模式 |
| 搜索所有 | `Cmd + Shift + F` | 全文搜索 |
| 插入链接 | `[[` | Wikilink 自动补全 |
| 插入 Callout | `> [!NOTE]` | 标注框 |

### Claude Code 常用指令

```bash
# 批量生成文档
./regenerate_all.sh                  # 全部
./regenerate_all.sh --subset core    # 仅核心
./regenerate_all.sh --subset marketing # 仅营销

# 同步 Memory → Obsidian
bash 脚本/sync_memory_to_obsidian.sh

# 从 Obsidian 提取知识给 Claude
bash 脚本/obsidian_context_for_claude.sh model  # 模型知识
bash 脚本/obsidian_context_for_claude.sh brand  # 品牌知识

# Claude 产出 → Obsidian 笔记
echo "内容..." | bash 脚本/claude_to_obsidian.sh meeting "主题"
```

## Dataview 常用查询

### 查看最近7天活跃笔记
```dataview
TABLE file.mtime as "修改时间"
FROM "知识库"
WHERE file.mtime >= date(today) - dur(7 days)
SORT file.mtime DESC
```

### 查看所有会议纪要
```dataview
TABLE date as "日期", participants as "参会人员"
FROM "会议纪要"
SORT file.ctime DESC
```

### 按标签筛选
```dataview
LIST
FROM #core
SORT file.name ASC
```

### 待办任务总览
```dataview
TASK
FROM ""
WHERE !completed
LIMIT 20
```
