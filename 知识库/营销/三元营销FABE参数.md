# 三元营销 FABE 参数

## 三大营销工具

| 参数 | 代金券 | 积分 | 消费金 |
|------|--------|------|--------|
| **发放比例** | 5%（占交易额） | 1:100（消费¥1=1积分） | 平台配置 |
| **抵扣上限** | 30%（单笔） | 20%（单笔） | 30%（核销上限） |
| **有效期** | 90天 | 2年 | 12个月 |
| **通用性** | 🆕 全平台通用 | 🆕 全平台通用 | 🆕 全平台通用 |
| **资金来源** | 营销池 | 营销池 | 消费金池 |
| **核销场景** | 任意商家 | 任意商家 | 任意商家 |

## FABE 模型

### 代金券 (Coupon)
| 维度 | 内容 |
|------|------|
| **F**eature | 消费后自动发放，5%面额，30%抵扣 |
| **A**dvantage | 跨店通用，90天有效期促进复购 |
| **B**enefit | 消费者感知"省钱"，商家获得"回头客" |
| **E**vidence | 行业数据：代金券平均提升复购率 25-40% |

### 积分 (Points)
| 维度 | 内容 |
|------|------|
| **F**eature | 消费¥1=1积分，1:100兑换率，20%抵扣 |
| **A**dvantage | 2年有效期，全平台通用，不可兑现 |
| **B**enefit | 长期黏性工具，积分累积→权益升级 |
| **E**vidence | 会员积分制平均提升LTV 30-50% |

### 消费金 (Consumer Fund)
| 维度 | 内容 |
|------|------|
| **F**eature | 平台发放/活动获取，30%核销上限 |
| **A**dvantage | 12个月有效期，全平台通用 |
| **B**enefit | 用户感知"免费额度"，强拉新+促活 |
| **E**vidence | 消费金核销率 60-80%，带动关联消费 |

## 跨店通兑结算

```
消费者在A店获得代金券 → 在B店消费使用代金券
→ B店收银系统核销代金券
→ 汇付结算：营销池 → B店（补差金额）
→ A店不受影响（营销池统一承担）
```
