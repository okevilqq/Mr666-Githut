# 链邦赋商通 · 品牌定位

## 一句话定位

**面向社区商业的数字经营平台** — 商户独立经营 · 生态会员互通 · 消费权益流转 · 真实交易激励

## 三大目标

1. **帮助商家提升复购率** — 通过跨店通兑+会员权益互通，让消费者在生态内持续消费
2. **帮助用户获得持续消费权益** — 代金券/积分/消费金全平台通用，消费即权益积累
3. **帮助社区形成可持续商业循环** — 本地商家→消费者→推广者，三方共赢

## 品牌名含义

**链邦赋商通** = 链（连接）+ 邦（社区）+ 赋（赋能）+ 商（商业）+ 通（通达）

- **链邦**: 连接社区商业生态
- **赋商**: 赋能商家经营能力
- **通**: 全域通达、消费流转

## 品牌核心概念

| 概念 | 含义 |
|------|------|
| 消费流转 | 消费者在生态内的自然流动路径 |
| 权益互通 | 在A店获得、在B店使用 |
| 千面千店 | 三类商家差异化呈现 |
| 交易即结算 | 零固定费用、零预充值、零资金占用 |

## 品牌主张

> **帮助商家经营，而非替代商家**
